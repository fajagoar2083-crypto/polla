## Funciones

- Inicio de sesión con PIN de 4 dígitos.
- Sin PIN correcto no se pueden modificar pronósticos.
- Admin puede eliminar participantes y cambiar PIN.
- Primera fase visible desde el inicio.
- Segunda fase se habilita desde Admin cuando termine la primera fase.
- Una vez iniciado el primer partido de treintaydosavos/ronda de 32, el admin bloquea la segunda fase.
- Al bloquear la segunda fase nadie puede modificar pronósticos desde ronda de 32 hasta la final.

## Puntajes

- Fase de grupos: 3 puntos
- Treintaydosavos / Ronda de 32: 6 puntos
- Octavos de final: 7 puntos
- Cuartos de final: 8 puntos
- Semifinales: 9 puntos
- Final: 10 puntos
