# Polla Mundialista 2026 - Mi Bracket FIX navegación

Esta versión corrige el error que hacía que no funcionaran los botones de navegación como Inicio, Registro e Iniciar sesión.

## Incluye

- Pestaña Mi Bracket.
- Eliminatorias sin empate.
- Registro con botón bloqueado al primer clic.
- Confirmación al final.
- No requiere cambiar Apps Script.
- No borra participantes ni pronósticos de Google Sheets.

## Antes de subir

Abre `index.html` y pega tu URL de Apps Script en:

```js
const APPS_SCRIPT_URL = "PEGA_AQUI_TU_URL_DE_APPS_SCRIPT";
```
