# Polla Mundialista

App web para una polla mundialista con registro, pronósticos, ranking, premios y bloqueo por fases.

## Reglas principales

- Primera fase: todos los pronósticos se hacen antes de iniciar el Mundial.
- Durante la fase inicial no se puede modificar nada.
- Al terminar la fase inicial, el administrador habilita la segunda fase cuando ya estén los clasificados.
- La segunda fase queda bloqueada antes del primer partido de esa fase.
- Cada acierto suma 3 puntos.

## Publicar en Vercel

Configuración recomendada:

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install
