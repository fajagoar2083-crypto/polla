# Polla Mundialista 2026 - Eliminatorias sin empate V2

Esta versión corrige la visualización de opciones:

- En fase de grupos (`fase1`) aparecen:
  - Gana local
  - Empate
  - Gana visitante

- En fase eliminatoria (`fase2`) aparecen únicamente:
  - Clasifica local
  - Clasifica visitante

## Importante

Si todavía ves Empate en fase eliminatoria después de subir esta versión:
1. Asegúrate de subir este `index.html`.
2. Haz Commit changes en GitHub.
3. En Vercel espera a que quede Ready.
4. Abre la app con Ctrl + F5.
5. Si estás en primera fase, es normal que salga Empate.
