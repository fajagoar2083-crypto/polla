# Polla Mundialista 2026 - Google Sheets

Esta versión usa Google Sheets como base de datos para máximo 30 participantes.

## Antes de subir a GitHub

1. En Apps Script pega el código de `apps_script_codigo_actualizado.txt`.
2. Implementa como aplicación web.
3. Copia la URL que termina en `/exec`.
4. Abre `index.html`.
5. Busca esta línea:

```js
const APPS_SCRIPT_URL = "PEGA_AQUI_TU_URL_DE_APPS_SCRIPT";
```

6. Reemplázala por tu URL de Apps Script.

## Archivos para GitHub

- index.html
- copa-mundial.svg
- README.md

El archivo `apps_script_codigo_actualizado.txt` NO se sube a GitHub. Solo sirve para copiar el código en Apps Script.
