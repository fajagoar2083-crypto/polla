# Polla Mundialista 2026 - PIN de usuarios

Archivos incluidos:

- index.html
- copa-mundial.svg
- README.md

## Funciones

- Cada participante crea un PIN de 4 dígitos al registrarse.
- Para entrar a sus pronósticos, debe seleccionar su nombre y escribir su PIN.
- Sin PIN correcto no puede modificar ni confirmar pronósticos.
- El administrador puede cambiar el PIN de un usuario si lo olvida.
- Admin puede eliminar participantes.
- Clave admin: Fabio2026.
