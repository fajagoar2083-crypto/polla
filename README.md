# Polla Mundialista 2026 - Pronósticos solo grupos + Mi Bracket

Esta versión deja la app organizada así:

- **Pronósticos**: solo fase de grupos.
- **Mi Bracket**: segunda fase completa.

## Importante

Este cambio NO toca Google Sheets, por eso:

- No borra participantes.
- No borra pronósticos.
- No borra pagos.
- No borra partidos.
- No requiere cambiar Apps Script.

Solo cambia el archivo `index.html`.

## Antes de subir

Abre `index.html` y pega tu URL de Apps Script en:

```js
const APPS_SCRIPT_URL = "PEGA_AQUI_TU_URL_DE_APPS_SCRIPT";
```
