# Polla Mundialista 2026 - Inicio de sesión con PIN

Archivos incluidos:

- index.html
- copa-mundial.svg
- README.md

## Funciones

- Cada participante crea un PIN de 4 dígitos al registrarse.
- Cada participante debe iniciar sesión con su nombre y PIN.
- Solo el usuario con sesión activa puede modificar sus pronósticos.
- Admin puede cambiar el PIN si alguien lo olvida.
- Admin puede eliminar participantes.
- Clave admin: Fabio2026.
