# Polla Mundialista 2026 - Google Sheets rápida final

Cambios incluidos:

- El botón **Crear mi participación** se bloquea al primer clic para evitar registros duplicados.
- El botón **Guardar y confirmar mis pronósticos** aparece al final, después de todos los partidos.
- Las selecciones se hacen rápido en pantalla.
- Solo se guardan en Google Sheets cuando el usuario confirma la fase.

## Apps Script

Si ya tienes funcionando el código rápido de Apps Script, no necesitas cambiarlo.

## index.html

Pega tu URL de Apps Script en:

```js
const APPS_SCRIPT_URL = "PEGA_AQUI_TU_URL_DE_APPS_SCRIPT";
```
